# 🎬 Kahani Shorts — Hindi Faceless AI Channel (youtube_shorts_bot)

**Goal:** Roz 1-2 Hindi suspense/kahani shorts → views + subscribers → monetize
**Content:** 100% AI original (script + image + voice) — koi copyright/strike nahi ✅
**Budget:** ₹0 (sab free tools)
**Edge:** Bot se 80% automate (Pradeep — bot maker)
**Length:** har short 30-45 sec (best retention)

## 📖 Zaroori files (padho)
- [POLICY.md](POLICY.md) — ⚖️ YouTube rules + har short se pehle checklist (channel safe rakho)
- [USED-IDEAS.md](USED-IDEAS.md) — 🚫 duplicate tracker (koi kahani repeat na ho)

---

## 📁 Folder Structure

```
kahani-shorts/
├── README.md              ← ye file
├── POLICY.md              ← YouTube T&C checklist
├── USED-IDEAS.md          ← duplicate tracker
├── scripts/               ← har short ki kahani + prompts + upload details
├── assets/ (images/voice/music)  ← AI se bani files (git me nahi jaati)
├── output/                ← kaam chal raha
└── ready-to-upload/       ← final videos
```

---

## 📋 Daily Workflow (30-40 min)

1. POLICY.md checklist dekho + USED-IDEAS.md me duplicate check
2. `scripts/` se ek kahani lo (ya nayi banwao)
3. 5 image prompts → Bing → images → `assets/images/`
4. Voice script → AI voice → `assets/voice/`
5. CapCut: images + voice + subtitle + music → `ready-to-upload/`
6. YouTube upload (title/desc/tags script file me hain)

---

## 🎯 Rules

- ✅ Sirf **original AI content** — kisi ka clip/cartoon copy MAT karo
- ✅ Hook pehle 3 second me strong
- ✅ 30-45 second shorts
- ✅ Twist ending
- ✅ Roz consistency (3-4 mahine grow hone me)
- ✅ Har short se pehle [POLICY.md](POLICY.md) checklist
